# Axway zlib

This library for generate pdf in axway analytics

## How To Use

* Extract Axway-zlib.zip or `git clone https://github.com/RizkiMufrizal/Axway-zlib.git`
* Access the directory and run `./configure`
* run `make` for compile the source code
* copy `libz.so.1` to `<Axway-Instalation/analytics/platform/lib`
